# Projeto - Clone do instagram 

# Objective
* Learn more about Ruby on Rails and some features

# Functionalities
* posts
  * creating posts with comments and photos
  * likes and dislikes on posts
  * comments on posts
  * Show comments from the user and others

  * user creation
  * password recovery
  * user creation
  * log out
